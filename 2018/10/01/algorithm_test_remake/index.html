    문제 설명

    히스토그램 만들기
    주어진 float 값 들을 히스토그램으로 만들려고 합니다.
    Float list “a” 가 주어지고, bin의 개수 “n” 이 주어졌을 때, 해당 bin에 몇 개의 값이 들어가는지 보여주는 histogram bin 내의 숫자를 카운트하는 함수를 만들어야 합니다.

    • Ex1) a = [1.0, 1.5, 1.7, 2.5, 3], n = 2 => [3, 2]
    • Ex2) a = [2.0, 2.1, 2.2, 2.3], n = 2 => [2, 2]
    • Ex3) a = [2.3, 2.2, 2.0, 2.1], n = 2 => [2, 2]

    참고사항

    1) 예외상황(n=0이된다거나,a가비어있다거나)은없다고가정하고,예외처리는크게 신경쓰지 않으셔도 됩니다.

    2) 그 외의 예외 상황 (a=[1,2,3] 인데 n=2 같은) 등의 예외적인 케이스는 특정 기준을 임 으로 정해서 처리하거나, 없다고 가정하셔도 됩니다.

    3) 프로그래밍 언어에는 제약이 없습니다. 다만 추가 라이브러리는 최대한 사용하지 않고 프 로그래밍 언어 내에서 제공하는 기본 라이브러리를 최대한 활용해 주세요.

    4) 가급적 OS에 의존적인 코드는 줄여 주세요. (예를 들어 C++ 로 개발 시 windows나 linux 에 dependent 한 코드라던지. 그런 경우 os별 호환 코드도 추가 필요)
