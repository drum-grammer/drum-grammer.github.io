1. What is Big O ?

Big O는 알고리즘의 성능을 수학적으로 표현해주는
점근 표기법의 한 종류이다.
그렇다면 점근 표기법이 무엇인지 알아보자:

1) 점근 표기법(asymptotic notation)

어떤 함수의 증가 양상을 다른 함수와의 비교로 표현하는 수론과 해석학의 방법이다.
알고리즘의 복잡도를 단순화할 때나 무한급수의 뒷부분을 간소화할 때 쓰인다.

대표적으로 다음의 다섯 가지 표기법이 있다.

  • 대문자 O 표기법
  • 소문자 o 표기법
  • 대문자 오메가(Ω) 표기법
  • 소문자 오메가(ω) 표기법
  • 대문자 세타(Θ) 표기법

타 표기법에 대해서는 아래 블로그 글 및 동영상 참조
https://ratsgo.github.io/data%20structure&algorithm/2017/09/13/asymptotic/
https://www.youtube.com/watch?v=Chcl71vEkRg&list=PL3U8xe-FeV8jUwc2Um1CCu4kkcKCVqDyi
이 중 가장 많이 쓰이는 것이 최악의 상황을 메인 펙터에 대해서 표기하는
대문자 O 표기법이다.

특히,알고리즘의 복잡도를 나타내는 용어로는
“계산 복잡도 이론” 또는 “시간복잡도”로
대문자 O 표기법이 일반적으로 사용된다.

2) 대문자 O 표기법 정의

2. 종류

1) O(1) [constant time]

종류:
- 배열 색인 액세스 (int a = ARR [5];)
- 연결된 목록에 노드 삽입
- 스택에서 푸시 및 팝하기
- 대기열에서 삽입 및 제거

예제코드1>

데이터의 증가와 상관없이 상수 시간 만큼의 처리시간이 필요한 알고리즘

2) O(n) [linear time]

종류:
- 배열 탐색
- 문자열 비교

예제2>

처리시간이 데이터의 증가량과 비례하는 경우

3) O(logn)

종류:
- 바이너리 검색
- 이진 검색 트리에서 최대 최소 값 찾기
- 특정 분할 정복 알고리즘

예제6> binary search

4) O(nlogn)

종류:
- 병합정렬
- 힙 정렬
- O(n^2) 알고리즘 최적화에 기반한 분할 정복 알고리즘

5) O(n^2) [quadratic time]

종류:

  • 버블 정렬
  • 삽입 정렬
  • 선택 정렬
  • 간단한 2D 검색

6) O(n^3) [polynomial / cubic time]

7) O(2^n) [exponential time]
