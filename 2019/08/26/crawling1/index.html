1. Selenium 이란?

공식 홈페이지
공식 홈페이지 정의에서도 볼 수 있듯이, Selenium 은 일종의 Web automantion browser이다.
이것을 사용하여 응용하는 분야는 사용자가 어떻게 활용하는냐에 달려 있다.

본 application 에서는, Selenium을 사용하여, 웹 환경에서 크롤러가 브라우저 처럼 로그인하고 데이터에 접근할 수 있도록 하는 방법에 대해 연구해볼 것이다.
